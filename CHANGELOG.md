# Changelog

本文件记录仓库的重要变更。

格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，版本号语义可按需逐步规范。

## [Unreleased]

### Added
- 预留：记录下一阶段尚未发布的功能与调整。
- 新增前端轻量国际化模块，支持中文与英文界面切换，并在首页右上角提供语言切换胶囊入口。
- 新增后端响应国际化适配层，可根据 `Accept-Language` 返回中英文 message。

### Changed
- 导航管理支持公共导航分组与网址的启用 / 停用，删除降级为次级危险操作。
- 个人导航移除启用 / 停用控制，回归为编辑 / 删除的轻量管理方式。
- 管理页中停用的公共导航分组与网址增加更明显的灰显样式，降低与启用状态的混淆。
- 管理页公共网址列表的操作列固定到整行最右侧，并将表头“操作”文字同步右对齐。
- 首页、管理页、登录页、注册页、设置页的核心文案已替换为国际化语言 key。
- 前端 API 请求现在会自动携带当前语言到后端，保证接口返回文案与页面语言一致。
- 修复设置页头像上传原生 input 直接显示的问题，改为仅通过按钮触发文件选择。
