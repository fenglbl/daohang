# 导航站项目

一个支持公共导航与私有导航的导航站项目。

## 项目结构

- `web/`：前端，基于 Vue 3
- `backend/`：后端，基于 Node.js + Express + MySQL

## 当前能力

- 用户注册 / 登录 / 获取当前用户 / 退出登录
- 导航分组 CRUD
- 导航链接 CRUD
- 公共导航与私有导航基础能力
- 管理员与普通用户分流登录
- 搜索引擎管理（支持启用 / 停用）

## 技术栈

### 前端
- Vue 3
- Vite

### 后端
- Node.js
- Express
- MySQL

## 运行说明

### 1. 安装依赖
分别在前后端目录安装依赖：

```bash
cd web
npm install

cd ../backend
npm install
```

### 2. 配置环境变量
在 `backend` 中配置数据库连接等环境变量。

### 3. 启动项目
根据前后端各自的脚本启动开发环境。
